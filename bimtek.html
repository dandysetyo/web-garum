Bimtek Admin Web Se-Kabupaten Blitar

Dinas Komunikasi dan Informatika • Mei 8, 2019 • Berita

Blitar-admin web Kecamatan Garum mengikuti Bimtek peningkatan kapasitas melalui Dinas Kominfo Kab.Blitar bertempat di ruang perdana Jl,sudanco supriyadi no.17 Kota Blitar padad rabu(8-5-2019)

Kegiatan ini dilaksanakan di ruang Perdana, Jalan Sudanco Supriyadi Nomor 17, Kota Blitar pada Rabu, 8 Mei 2019.

Kegiatan Bimtek ini diikuti oleh 22 kecamatan yang ada di wilayah Kabupaten Blitar,Masing-masing admin web kecamatan mendapatkan materi tentang tata cara posting ke web dari Dinas Kominfo.(KSM-Garum)
